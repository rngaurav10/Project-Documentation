1. Project Overview

1.1 Purpose and Justification

The project will create a centralized SQLite/SQLAlchemy database in Python to consolidate data from SAP, Cognos, MicroStrategy, and Salesforce. It will automate ETL processes, reduce manual effort, and serve as a single source of truth for Sales, Distribution, and Partner Management.

1.2 Objectives

  • Build centralized database with input, mapping, and output tables.
  • Automate ETL using Python.
  • Consolidate SAP, Cognos, MicroStrategy, Salesforce extracts.
  • Deliver standardized Allianz, Vermittler, Banken, Broker reports.
  • Provide ad-hoc query support.
  • Enable Power BI dashboards for analysis.

1.3 Business Benefits

  • â‰¥40% reduction in manual reporting effort.
  • Improved KPI consistency.
  • Flexibility for ad-hoc requests.
  • Transparency via shared data definitions.

2. Scope Management

2.1 In Scope
